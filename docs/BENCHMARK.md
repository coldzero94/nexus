# NEXUS 기능 벤치마크 — 베낄 것 vs 피할 것

기준일 2026-07-13. 전제: MVP = 서버 없는 iOS 로컬 앱. 서드파티 추정치는 (추정) 표기. 출처·수치 신뢰도는 [RESEARCH.md](./RESEARCH.md) §9 참고.

---

## 1. 온보딩·권한 요청

**베낄 것**
- [MVP] **Finch — 가입 전 캐릭터 커스터마이징부터**: 권한 요청 전에 캐릭터 꾸미기로 첫날 애착 형성. NEXUS 순서: 캐릭터 생성 → 첫 걸음이 XP가 되는 순간 체험 → 그다음 HealthKit 권한. 서버가 없어 계정 생성 자체가 불필요한 것이 오히려 온보딩 무기.
- [MVP] **Pokémon GO Adventure Sync — 자동 수집이 대중화의 전제**: 이 카테고리 승자는 전부 "앱을 안 켜도 쌓이는" 구조. 수동 기록을 요구하는 단계를 하나도 만들지 말 것.
- [MVP] **운동→스탯 매핑은 자동 기본값**: 매핑(걷기→지구력 등)은 앱이 결정, 사용자는 첫 화면에서 읽기만.
- [MVP] **런데이 — 커리큘럼형 초반 구조**: '8주 프로그램' 단계형 설계가 국내 설치 1위를 만들고 여성 입문자를 흡수. 초반 퀘스트를 주차별 성장 커리큘럼 + 입문자 친화 톤으로.
- [MVP] **권한 거부 내성(Apple 5.1.1(ii))**: 거부 시에도 기본 사용 가능 + 데모 모드.

**피할 것**
- **Habitica — 시스템 이해가 온보딩을 잡아먹음**: 비게이머가 "습관 관리보다 게임 이해에 시간을 씀". 첫 화면에서 설명 없이 이해 안 되면 실패.
- **Do It Now/LifeUp — 설계 부담의 사용자 전가**: 스탯 체계를 사용자가 직접 만들게 해 파워유저 니치에 갇힘.
- **Peridot — 고마찰 첫 세션**: AR 카메라 필수 → Niantic도 3년 만에 종료.

## 2. XP·성장 산식

**베낄 것**
- [MVP] **Pikmin Bloom — 걸음이 '자라는 모습'으로 즉시 시각화**: 운동 데이터는 반드시 캐릭터의 외형·능력치·히스토리에 직접 새길 것.
- [MVP] **운동 '종류'의 능력치 분화**: '걸음→RPG'는 150만 DL(Shikudo Fitness RPG)로 수요 검증됐으나 걸음 단일 자원에 머묾 — 3축 분화가 NEXUS 고유 영역.
- [MVP] **NRC/Strava Local Legend — 속도가 아닌 총량·꾸준함 보상**: 빈도(=리텐션) 자체를 보상. 초보자가 이길 수 있는 게임으로.
- [MVP] **Pokémon Sleep — '정산 개봉' 의식**: 운동 종료 직후 워치에서 XP 정산 애니메이션 개봉 = NEXUS판 '아침 개봉'.
- [MVP] **Apple 활동 링 — 개인 이력 기반 적응형 목표**: 절대 기준이 아닌 '나의 지난 4주 대비'가 초보·상급자 모두에게 공정.
- [MVP] **트랭글의 역설 — 환산 투명성**: 게임화 원조가 데이터 불신으로 평점 2.5까지 추락. "왜 이 운동이 이 XP인가"를 반드시 보여줄 것.

**피할 것**
- **Wokamon/Walkr — 걸음=단일 화폐 환전**: 2주면 성장 의미 소진, 애착 실패.
- **Shikudo Fitness RPG — 경제 붕괴**: 레벨 38에 화폐 12종. XP + 소수 능력치로 극단적으로 단순하게.
- **Habitica — 검증 없는 자기신고 경제**: 센서 실측이 이 결함을 제거한다는 점을 마케팅에도 쓸 것(단 '조작 불가' 문구는 금지 — RESEARCH.md §7.3).
- **STEPN/슈퍼워크 — 환금성 보상**: 보상이 시장 가격에 연동되면 리텐션도 시장에 연동됨. 어떤 단계에서도 금지.
- **Do It Now — 숫자만 오르는 성장**: 능력치 상승은 시각 변화(외형·장비·전직)를 반드시 동반.

## 3. 스트릭·복귀 루프

**베낄 것**
- [MVP] **Duolingo — 용서 장치의 정량 근거**: 스트릭 프리즈만으로 DAU +0.38%, 평균 스트릭 30.6일 vs 18.9일. '휴식도 정상 보상'을 정당화. 단 잃을 게 아예 없으면 복귀 동기도 없음 → **'컨디션/기세' 같은 소프트 손실 대상(하락은 있되 소멸 없음)** 필수.
- [MVP] **Finch — 죄책감 제로 재진입**: 며칠 쉬다 와도 반겨줌. 복귀 환영 연출을 1급 기능으로.
- [MVP] **Finch — 8시간 어드벤처 appointment**: 운동으로 모은 에너지로 원정 보내고 몇 시간 뒤 결과 개봉 → 하루 2~3회 자연 재방문. 로컬 타이머만으로 구현 가능.
- [MVP] **Pokémon Sleep — 아침 리추얼**: 기상 후 "간밤의 회복 + 어제의 성장 요약" 카드.
- [MVP] **Pikmin Bloom — 안 열어도 손해 없는 루프 + 하루 요약 카드**: '오늘의 모험 일지'로 차용.
- [MVP] **Duolingo — 알림 규율**: 푸시 일 최대 2건. 그대로 복사 가능한 원칙.
- [MVP] **월 1회 한정 배지 캘린더**: 저비용 고효율(Apple/Garmin 검증). 단 캐릭터에 영구히 남는 수집 자산 형태로.

**피할 것**
- **Apple 활동 링 — 휴식 없는 스트릭 강박**: 최대 불만. "링은 다그치지만 NEXUS 캐릭터는 쉬는 날 함께 쉰다"가 포지셔닝 공간.
- **Habitica — 처벌(HP·죽음) 루프** / **Forest — 처벌의 적용 한계**(짧은 계약에서만 작동) / **Nomi·TameWatch — 펫 사망 설계**(이 카테고리 최다 불만이 전부 사망 메커닉).
- **Finch의 '3개월차 숙제화'**: 자동 기록이라 구조적으로 면역 — '체크리스트 없는 다마고치'를 핵심 메시지로.

## 4. 캐릭터 정서 설계

**베낄 것**
- [MVP] **Finch — 동기의 외재화**: "나를 위해선 못 해도 내 캐릭터를 위해선 한다". 온보딩 첫 3분 안에 "이 캐릭터는 내 운동으로만 자란다"는 의존 관계 각인.
- [MVP] **Kinder World 대조 — 반응하는 캐릭터**: 같은 무처벌 성장물인데 반응하는 새(Finch)와 정적인 식물(Kinder World)이 성패를 가름. 대화·리액션·개성(운동 후 반겨주기, 휴식일에 같이 뒹굴기) 필수.
- [MVP] **애착의 원천 = '내 운동 인생의 아카이브'**: IP 없는 오리지널 캐릭터의 한계는 애착을 외모가 아니라 '능력치로 시각화된 내 기록'에 두는 것으로 극복 — 쌓일수록 대체 불가능해지는 데이터 자산.
- [MVP] **아트 예산은 반응 다양성에 우선 배분**: Pokémon Sleep 이탈 원인이 '같은 애니메이션 소진'. 대사·표정·포즈 조합 다양성이 곧 리텐션.
- [후속] **Zombies, Run! — 운동 중 실시간 피드백**: 워치 세션 중 햅틱+짧은 응원 연출로 응용(오디오 드라마 제작은 2인 팀에 불가).

**피할 것**
- **Aloe Bud — 다정한 톤만, 자라는 존재 없음**(150만 DL에도 정체. 톤 ≠ 리텐션).
- **Peridot — 귀여움+기술력만**: 펫이 사용자의 자기 개선과 무관하면 "왜 계속 키우나"에 답 못 함.
- **바디프로필식 단기 변신 서사**: 타깃 세대 동기는 '꾸준함의 정체성화'로 이동 — 누적 성장 톤으로.

## 5. 워치·위젯 경험

**베낄 것**
- [MVP] **Finch 위젯 4대 장치를 그대로 요구사항화**: ① 살아있는 캐릭터 상태 상시 표시 ② 원정 '진행 중/완료' 상태 ③ 일일 진행바 ④ 아침/저녁 마이크로 이벤트. 위젯 애니메이션 불가 제약 안에서 살아있는 감각을 만든 대규모 검증 사례. Duolingo도 위젯 사용자의 리텐션 우위를 확인(자기선택 통제 후).
- [MVP] **Pixel Pals — '폰이 본체, 워치는 얇은 창'**: 다이나믹 아일랜드 Live Activity + 잠금/홈 위젯 + StandBy. 워치는 Live Activity 자동 미러링(watchOS 11+)으로 따라오게.
- [MVP] **HKWorkoutSession = 유일한 실시간 애니메이션 구간**: "XP가 차오르는 캐릭터와 함께 달리는" 킬러 신.
- [MVP] **RelevanceKit 선점**: 운동 직후·기상 직후 캐릭터가 스마트 스택 맨 위로 마중.
- [MVP] **컴플리케이션 = 시간대별 정지 스냅샷**: 일 ~40-75회 갱신 예산으로 충분.
- [MVP] **런데이 — 워치 과금 검증**(연 29,000원): 워치 완성도 자체가 차별화이자 과금 근거.

**피할 것**
- **'워치페이스에 캐릭터가 산다' 문구**: watchOS에서 기술적으로 불가 — 기대 불일치 리뷰 역풍.
- **Pex — 폰-워치 동기화 버그**(카테고리 1순위 품질 불만): HealthKit을 단일 진실 소스로, 양쪽이 각자 계산.
- **Nomi — 워치 전용 + 방치**: 워치 레이어는 유지비 낮은 WidgetKit+운동 화면 중심으로 얇게. 게임 로직·꾸미기는 폰에.
- **iCloud 원본 건강 데이터 동기화**(Apple 5.1.3(ii) 리젝): 동기화는 파생 게임 상태만.

## 6. 커뮤니티·모임 (전부 후속 단계)

**베낄 것**
- **당근 — 순서: 단독 가치 먼저, 커뮤니티는 나중**: MVP에서 커뮤니티를 안 만드는 것 자체가 이 패턴의 반영. (러닝크루 매칭 신생 앱군은 유저 밀도 없이 매칭부터 시작해 집단 콜드스타트 실패 중.)
- **소모임 — 리더 과금·멤버 무료**(월 15,500원, 12년 생존): 2인 팀에 가장 현실적인 커뮤니티 수익 구조. 참가비 중개 시 문토 '수수료 20%'가 시장 표준.
- **트레바리식 스킨인더게임의 무부담 버전**: 워치 자동 기록이 '독후감' 역할 — "주간 XP 달성자만 크루 시즌 이벤트 참여" 같은 자격 조건을 게임 메커닉으로(사진 인증·벌금 불필요). ※ '조작 불가' 전제는 과장이므로 신뢰 등급(Tier A/B/C) 설계 병행.
- **넷플연가·버핏서울 — 시즌제 커밋먼트**: 3개월 시즌 + 시즌 한정 캐릭터 보상.
- **'밴드+카톡+인스타' 조합의 구멍만 공략**: 워치 데이터 자동 출석, 크루 XP 리더보드, 캐릭터 대항전. 소통은 오픈채팅 연동으로 얹혀가고 밴드 대체 시도 금지.
- **Strava/Duolingo — 코호트 경쟁**: 전역 1등이 아닌 '내 동네·내 수준' 상대 성취, 유사 참여도 코호트 리그. RCT 메타분석도 '경쟁이 협력보다 일관되게 우위'.
- **워크온 — B2G/B2B 경로**: 지자체·기업이 걷기 챌린지에 이미 예산 집행(유료 기관 1,000+). 길드 대항전을 스폰서 챌린지로 파는 검증된 경로.

**피할 것**
- **남의집 — 모임 중개 단독 수익화 실패**(당근 투자에도 폐업): 커뮤니티는 수익원이 아니라 리텐션 엔진.
- **Habitica 파티 — 타인의 실패가 나의 손해**: 어떤 단계에서도 금지.
- **Fitbit — 소셜 기능의 후행 철회**("그게 유일한 이유였다"는 대탈출): 축소가 필요 없는 최소 단위(친구 캐릭터 보기)부터.
- **Meetup — 오거나이저 과금 강화로 신뢰 붕괴**: 크루장에겐 초기에 과금 대신 혜택(전용 아이템·크루 커스터마이징).
- **Strava 한국 철수(2025-03)의 경고**: 지역 기능 출시 전 LBS 신고와 민감정보 동의 설계를 기능 기획보다 먼저.

## 7. 수익화

**베낄 것**
- [MVP] **경계는 출시일에 긋는다 — 무료 코어 + 소프트 페이월**: Finch(공식 월 $9.99/연 $69.99, 무료로 핵심 루프 전부 가능)가 벤치마크. 한국 가격 감각은 월 6,900~9,900원, 이 니치 검증 구독가는 연 ₩39,000~79,000(Gentler Streak 한국가).
- [MVP] **코어 루프 영구 무료 + 코스메틱·심화 과금**: 성장 속도가 아니라 '기록의 깊이(상세 스탯·히스토리) + 꾸미기 + 클래스 스킨'을 판다. Finch의 구독 동기가 '미안해서'가 아니라 '더 잘해주고 싶어서'라는 순양성 감정 설계.
- [MVP] **LifeUp/Streaks — 로컬 앱 + 일회성 프리미엄의 실증**: 서버 없는 모델과 궁합이 좋음. 구독과 병행 검토(Pixel Pals 평생 $49.99 티어 참고).
- [MVP] **규제 정합 포지셔닝**: '건강 및 피트니스' 카테고리 + 가챠 없음 + 운동→XP는 5.1.3(i) 부합(상세: RESEARCH.md §8).
- [MVP] **기대치 캘리브레이션**: 플랜핏 글로벌 누적 ~$5M (추정)이 현실적 상한 벤치마크. 목표는 '작지만 애착 깊은 유저의 높은 전환'.
- [후속] **챌린저스·워크온 — 브랜드 제휴/B2G 챌린지**: 앱 내 직접 과금의 천장을 보완.

**피할 것**
- **TameWatch — 운동의 대가를 구독 뒤에 잠금**(평점 2.0): 걸음·운동에서 나오는 XP·기본 보상은 절대 페이월 뒤에 두지 말 것.
- **Forest/Garmin — 무료였던 것의 후행 유료화**: 신뢰 파괴. 전환율 현실치는 2~8%(Duolingo ~9%가 최상급).
- **블록체인·토큰·현금성 리워드**: 반복 실패한 지뢰밭 + 법적 충돌(게임산업법 환전금지).
- **캐시워크류와의 현금 리워드 경쟁**: 광고 인프라 거인들의 레드오션. 이들이 학습시킨 행동만 넘겨받을 것.
- **광고 모델 자체**: 건강 데이터 기반 타겟팅 광고는 Apple 5.1.2(vi) 원천 금지 — 구조적으로 궁합 나쁨.
