# Project NEXUS

> **"체크리스트 없는 다마고치 — 당신이 움직이면 캐릭터는 알아서 자랍니다."**

폰·워치의 건강 데이터(걷기·러닝·근력)를 2D 캐릭터의 XP·능력치·클래스로 자동 전환하는 라이프 RPG. 캐릭터는 죽지 않고, 휴식도 성장이다. **Android 퍼스트** (iOS는 코어 루프 검증 후 확장).
